【C++ Primer】学习笔记2

字符串、向量和数组

1.标准库类型string

string表示可变长的字符序列

1.1定义及初始化string

(1)string s1; 默认初始化为空字符串
(2)string s2(s1) 创建s2作为s1的副本
(3)string s2 = s1 同上
(4)string s2("abcd") 创建s2作为字面值”abcd”的副本
(5)string s2 = "abcd" 同上
(6)string s2(n,"c") 创建一个字符串并初始化为n个c组成
直接初始化与拷贝初始化
上述用( )初始化的为直接初始化，用=初始化的为拷贝初始化。当初始值只有1个的时候，用二者皆可，但例如第六种情况，初始值有许多个的时，通常只能用直接初始化。

1.2 string的操作

(1)os << s 将s写到输出流os中，返回os
(2)is >> s 从is中读取字符串赋于s，字符串以空白分割，返回is
(3)getline(is,s) 从is中读取一行赋于s，返回is，一直读直到读取到换行符
(4)s.empty()
(5)s.size()
(6)s[n] 返回s中第n个字符的引用
(7)s1+s2 返回s1与s2连接结果
(8)s1 = s2 用s2的副本代替s1原来的字符
(9)s1 == s2 判断两个字符串是否全部字符相同
(10)s1 != s2
(11)<,<=,>,>=
注意的点：
1.cin 与 getline读取方式的不同
is >> s中，例如cin >> s;会读取输入中的内容，忽略开头的空白，从第一个真正字符开始读取，直到下一个空白字符出现。例如输入一个” Hello world”，这时候s的值为”Hello”。如果我们想要将空白符也读取进去，那则需要用getline读取。getline(cin,s); s的值就为” Hello world”。
2.string::size_type 类型
这个类型是一个无符号类型，是string::size()返回的类型，需要用无符号数接收。
3.string的比较问题
比较原则如下：
(1)如果字符串的长度不同，但前面部分相同，则长度长的为大。
(2)如果两个字符串长度从0开始，到后面存在不同的字符，则以这个字符按照字典顺序区分字符串的大小。
4.string和字面值相加
标准库中允许将字符串字面值、字符字面值转化为string对象。但是，string对象和字符字面值及字符串字面值混合在一条语句使用时，+两侧必须要有一侧为string对象。

1.3 处理string对象的字符

1.cctype头文件中有许多判断函数，能够对string中的字符进行判定，例如判定字符是否为数字，是否为空白，等等。在有需要的时候可以使用。
2.for循环
想要处理string中的每个字符，可以利用for循环。
语法形式：
for(declaration : expression)
statement
其中declaration用于初始化控制变量，用来访问基础元素，迭代后变量会被初始化为expression的下一个元素值。

输出字符串中的每个元素，其中c的类型是char，如果我们需要更改其中的类型，比如说将其中的所有都变成大写，则要写成auto &c : str
3.访问string对象的单个字符的两种方式：
(1)下标，下标返回的是对应位置的引用
(2)迭代器

2.标准库类型vector

vector表示对象的集合，其中所有对象类型相同。

2.1定义及初始化vector

(1)vector<T> v1，v1是空vector，类型为T，执行默认初始化。
(2)vector<T> v2(v1) ，创建v2为v1的副本
(3)vector<T> v2 = v1，创建v2为v1的副本
(4)vector<T> v3(n,val)
(5)vector<T> v5{a,b,c...}
(6)vector<T> v5 = {a,b,c...}
上述涉及三种方法，分别是列表初始化，创建指定数量的元素，值初始化。
列表初始化就是利用花括号，直接初始化vector中元素的值。值初始化是指只提供对象容纳的元素数量而略去初值(默认初始化，若为int，则默认为0)。
列表初始化时，花括号中的元素必须与元素类型相同，否则无法执行列表初始化，编译器会尝试用默认初始化vector对象。例如
vector<string> v8{10, "hi"}，其中10不是字符串，那么将表达为vector<string> v8(10,"hi") 。又如vector<string> v8{10}，则表达为vector<string> v8(10)。结果是10个空字符串组成的vector。

2.2vector对象的操作

(1)v.push_back() 向vector中添加元素
(2)v.empty()
(3)v.size()
(4)v[n]
(5)v1 = v2
(6)v1 = {a,b,c..}
(7)v1 == v2
(8)v1 != v2
(9)<,<=,>,>=

3.迭代器

在string和vector中，我们可以利用下标完成对元素的访问，迭代器也能实现相同的目的。标准库中只有少数支持下标，但全部支持迭代器。迭代器类似于指针类型，提供对象的间接访问。
C++程序员更愿意使用迭代器而非下标，因为迭代器在所有标准库中都通用。

3.1使用迭代器

与指针不同，迭代器的获取不是取址符，而是通过vector、string这样的类的成员函数返回，这些类型都拥有begin和end的成员函数，begin指向首元素，end指向尾元素的下一位置(注意不是尾元素)。
auto b = v.begin(), e = v.end();
如果我们不想更改容器内的值，可以声明const。如果对象本身就是常量，则返回常量迭代器。
auto const b = v.begin();
也可以利用cbegin和cend成员返回常量迭代器。
auto b = v.cbegin();
如此便定义了头迭代器和尾后迭代器。我们暂时不用在意迭代器的类型是什么。

3.2迭代器运算符

(1)*iter,返回指向元素的引用
(2)iter->mem，这种情况使用于iter指向的元素是一个类，用以访问类内成员
(3)++iter，令iter指向容器内下一个元素
(4)--iter
(5)iter1 == iter2 判断两个迭代器是否指向同一元素
(6)iter1 != iter2

4.数组

数组是一种类似于vector的数据结构，用于存放同数据类型数据，与vector不同的是数组具有固定的大小，不能随意增加。数组的元素应该为对象，因此不存在引用的数组。

4.1数组的定义和初始化

需要注意的是在声明数组时，大小必须是常量表达式，不可用变量生命数组。例如：
unsigned int a = 8;
int arr [a];
则会报错。

字符数组的特殊性
我们可以利用字符串字面值去初始化一个数组，但要注意字符串尾部的空字符也会储存到数组中。此时需要注意数组大小是否会溢出。
不允许拷贝和赋值
不能将数组的内容拷贝给其他数组作为初始值，也不能用其他数组给当前数组赋值。
理解复杂数组的声明
在理解复杂数组的声明时，可采用由内向外的阅读方式。例如：

strlen(p) 返回p的长度，空字符不算
strcmp(p1,p2) 比较p1和p2的相等性，若p1 == p2，返回0，如果p1 > p2 返回正值，若p1 < p2 返回负值。
strcat(p1,p2) 将p2附加到p1后，返回p1
strcpy(p1,p2) 将p2拷贝给p1,返回p1

string s1 = “abcd”;
string s2 = “bacd”;
if(s1 < s2) {…;}

const char ca1[] = “abcd”
const char ca2[] = “bacd”
if(ca1< ca2) {…;}

int ia[3][4];
size_t cnt = 0;
for(auto &row : ia)
for(auto &col : row){
col = cnt;
++cnt;
}

for(const auto &row : ia)
for(auto col : row){
cout << col << endl;
}

```
如果不加引用，那么row就会变成指向ia首地址元素的指针，但我们要得到的是ia中row的一维数组，才能进行遍历。
因此可以总结：
要使用for语句处理多维数组，除了最内层的循环外，其他所有循环的控制变量都必须是引用类型。
